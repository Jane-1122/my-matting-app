============================================
  Jing's Video Matting Studio - 使用说明
============================================

【项目简介】
  精细视频抠像网站，支持人物抠像和通用物品抠像，
  可上传视频、预览抠像效果、下载透明背景 PNG 序列 ZIP 包。


【前置条件】
  1. 安装 Docker Desktop
     下载地址: https://www.docker.com/products/docker-desktop/
  2. 安装后重启电脑
  3. 确保 Docker Desktop 已启动（任务栏有 Docker 图标）


【启动步骤】
  1. 解压项目压缩包到任意文件夹
  2. 双击 启动网站-Docker.bat
  3. 等待启动完成（首次运行需 5–15 分钟下载依赖）
  4. 浏览器会自动打开，或手动访问 http://localhost


【关闭步骤】
  双击 停止网站-Docker.bat 即可停止并关闭网站。


【常见问题】

  Q: 提示"未检测到 Docker"？
  A: 请先安装 Docker Desktop，安装后重启电脑。

  Q: 提示"启动失败"？
  A: 请确认 Docker Desktop 已运行（任务栏有 Docker 图标）。

  Q: 端口 80 被占用？
  A: 关闭占用 80 端口的其他程序，或修改 docker-compose.yml 中的端口映射。

  Q: 首次启动很慢？
  A: 首次需要下载镜像和构建，约 5–15 分钟，之后启动会快很多。
